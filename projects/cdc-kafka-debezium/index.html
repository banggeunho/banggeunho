CDC-Kafka-Debezium

Kafka Debezium CDC MySQL Java
View on GitHub

개요

CDC(Change Data Capture) 파이프라인을 Kafka와 Debezium을 활용하여 구축한 프로젝트입니다. 데이터베이스의 변경 사항을 실시간으로 캡처하여 다른 시스템으로 전파하는 이벤트 기반 데이터 동기화 시스템입니다.

주요 기능

  • MySQL binlog 기반의 실시간 변경 데이터 캡처
  • Debezium Connector를 통한 Kafka 이벤트 스트리밍
  • 데이터 일관성 보장을 위한 파이프라인 설계
