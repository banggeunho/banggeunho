Spring-Boot-MSA

Spring BootMSAGrafanaZipkinPrometheus
View on GitHub

개요

Spring Boot 기반의 마이크로서비스 아키텍처를 구현하고, Grafana, Zipkin, Prometheus를 활용한 통합 모니터링 체계를 구축한 프로젝트입니다.

주요 기능

  • Spring Cloud Gateway를 통한 API 게이트웨이
  • Eureka Service Discovery 기반 서비스 등록/탐색
  • Zipkin을 활용한 분산 트레이싱
  • Prometheus + Grafana 기반 메트릭 모니터링
